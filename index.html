What is qskeleton?

qskeleton is polyhedral computation software for solving the vertex and facet enumeration problems for convex polyhedra and polyhedral cones, and eliminating variables from systems of linear inequalities. It implements the double description method (DDM) and Fourier-Motzkin elimination (FME) with Chernikov rules.
